# SoundStudio Bot

Telegram‑бот для записи в звукозаписывающую студию с ролями клиент, звукорежиссер и администратор.

## Быстрый старт (локально)

1. Склонируйте репозиторий и перейдите в папку.
2. Скопируйте `.env.example` в `.env` и заполните переменные.
3. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python -m venv venv
   source venv/bin/activate   # Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```
4. Запустите бота:
   ```bash
   python bot.py
   ```

## Развёртывание на Railway

1. Создайте новый проект → **Deploy from GitHub** и выберите этот репозиторий.
2. Добавьте плагин **PostgreSQL** – Railway автоматически предоставит `DATABASE_URL`.
3. В разделе **Variables** добавьте:
   - `BOT_TOKEN` – токен от @BotFather
   - `ADMIN_IDS` – список Telegram‑ID админов через запятую
   - `TIMEZONE` – например, `Europe/Moscow`
4. Railway выполнит `worker: python bot.py` из `Procfile`. При первом запуске БД будет инициализирована.

## Возможности

- Бронирование 1 ч/2 ч без подтверждения, от 3 ч – с подтверждением.
- Шаг слота 30 мин, рабочие часы 11:00‑22:00.
- Доступно бронировать только на ближайшие два месяца.
- Выходные дни у звукорежиссёров.
- Ночная запись (требует ручного одобрения админа).
- Статусы записи: ожидает подтверждения, подтверждено, отменено, завершено.
- Уведомления: новая бронь, напоминание за 2 ч, запрос отзыва после завершения.
- Админ‑панель: управление пользователями, статистика, экспорт CSV, вкладки «Наша команда» (текст+фото) и «Контакты студии».
- Система отзывов (оценка 1‑5 + комментарий) после того, как запись получает статус «завершено».
- Все меню реализованы через инлайн‑кнопки, есть кнопки «Назад» и «Отмена».
